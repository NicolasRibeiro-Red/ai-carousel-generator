# PLAYBOOK BreathAI
## Guia Completo de Uso + Metodo Hormozi

---

## 1. PRIMEIRO USO (Calibracao)

Na primeira interacao com a BreathAI, faca a calibracao de estilo:

### Passo 1: Inicie a conversa
```
"Ola BreathAI, quero criar conteudo sobre Breathwork"
```

### Passo 2: Responda as perguntas de calibracao
A IA vai perguntar:
1. Qual seu tom preferido? (tecnico/inspirador/direto)
2. Usa emojis? (nenhum/poucos/muitos)
3. Cole 2-3 exemplos de posts seus

### Passo 3: Confirme o perfil
A IA criara um perfil de estilo. Confirme ou ajuste.

---

## 2. A REGRA DE OURO: HOOK PRIMEIRO

> "Depois de escrever sua manchete, voce gastou 80 centavos do seu dolar de publicidade." - David Ogilvy

**O hook e responsavel por 80% do sucesso do seu conteudo.**

### Formula do Conteudo Eficaz:
```
HOOK (Fisgar) → RETER (Manter) → RECOMPENSAR (Entregar)
```

### Antes de Criar Qualquer Conteudo:
1. Defina o HOOK primeiro (use os 8 tipos abaixo)
2. So depois crie o conteudo que cumpre a promessa
3. Teste pelo menos 3 variacoes de hook

---

## 3. OS 8 TIPOS DE HOOKS

Use estes tipos para criar hooks poderosos:

| Tipo | Formula | Exemplo Breathwork |
|------|---------|-------------------|
| **Rotulo** | "[Avatar], isso e pra voce" | "Ansiosos, tenho um presente pra voces" |
| **Pergunta Sim** | Pergunta obvia = sim | "Voce quer dormir melhor?" |
| **Pergunta Aberta** | Gera curiosidade | "Qual voce preferiria: estresse ou calma?" |
| **Condicional** | "Se X, entao Y" | "Se esta estressado, esta respirando errado" |
| **Comando** | Instrucao direta | "Leia isso se quer parar de ter ansiedade" |
| **Declaracao** | Afirmacao forte | "A coisa mais poderosa que voce pode fazer hoje..." |
| **Lista/Numero** | Numero + promessa | "5 respiracoes que mudam sua vida" |
| **Narrativa** | Historia | "Eu estava no auge da ansiedade quando..." |

### Prompt para Gerar Hooks:
```
"Sugira 10 hooks diferentes para este tema usando os 8 tipos:
- 2 Rotulos
- 2 Perguntas
- 2 Condicionais
- 2 Comandos/Declaracoes
- 2 Listas/Narrativas"
```

## 4. CRIANDO CONTEUDO

### Estrutura do Pedido
```
[FORMATO] + [TOPICO] + [OBJETIVO] + [TIPO DE HOOK]
```

### Exemplos de Pedidos (Metodo Hormozi)

**Carrossel com Hook de Lista:**
```
"Crie um carrossel sobre os 4 objetivos do Breathwork.
Hook tipo LISTA: '4 coisas que a respiracao pode fazer por voce'
Objetivo: educar iniciantes.
Use estrutura: HOOK → RETER (lista) → RECOMPENSAR"
```

**Reels com Hook de Comando:**
```
"Roteiro de Reels de 30s sobre tecnica 4-8.
Hook tipo COMANDO: 'Faca isso AGORA se esta ansioso'
Objetivo: viralizar.
Estrutura: Hook forte → Passos claros → CTA"
```

**YouTube com Hook de Declaracao:**
```
"Roteiro de 10min sobre CO2 e estresse.
Hook tipo DECLARACAO: 'A verdade sobre CO2 que ninguem te conta'
Objetivo: autoridade cientifica.
Use: Historia pessoal + Ciencia + Demonstracao"
```

## 5. FRAMEWORK 70-20-10 PARA HOOKS

Quando pedir hooks a BreathAI:

```
"Gere 10 hooks para este tema:
- 7 baseados em hooks que ja funcionaram (adapte estes exemplos: [cole exemplos])
- 2 variacoes criativas dos melhores
- 1 completamente experimental"
```

### Analise de Performance:
- **Top 10% dos seus posts:** Replique esses hooks
- **Bottom 10%:** Nunca use novamente
- **Meio:** Teste variacoes

## 6. TECNICAS DE RETENCAO

Peca a BreathAI para usar estas estruturas:

### Para LISTAS:
```
"Estruture como lista numerada.
Cada item deve gerar curiosidade para o proximo.
Exemplo: '7 tecnicas, sendo a #5 a mais poderosa'"
```

### Para PASSOS:
```
"Estruture como passo-a-passo.
Cada passo deve ser claro e acionavel.
'Passo 1... Passo 2... Passo 3...'"
```

### Para HISTORIAS:
```
"Conte uma historia com:
- Situacao inicial (problema)
- Conflito/descoberta
- Transformacao
- Licao aplicavel"
```

## 15. BOAS PRATICAS (Metodo Hormozi)

### DO (Faca)
- Gaste 80% do tempo no hook
- Teste 10x mais hooks que parece razoavel
- Replique 70% do que ja funcionou
- Use LISTA, PASSOS ou HISTORIA para reter
- Entregue mais do que prometeu
- Faca ELES se sentirem importantes
- Deixe os dados ensinarem

### DON'T (Nao Faca)
- Criar conteudo antes do hook
- Testar apenas 1-2 variacoes
- Inventar tudo do zero (use 70% comprovado)
- Falar de voce antes de falar deles
- Prometer mais do que entrega
- Ignorar o Top 10% e Bottom 10%
- Seguir intuicao em vez de dados

## 16. METRICAS E EVOLUCAO

### O Que Medir:
- Views/Impressoes por hook
- Engajamento por tipo de hook
- Salvamentos (indica valor real)
- Comentarios (indica conexao)

### Ciclo de Melhoria:
```
1. Publique 10 conteudos com hooks variados
2. Identifique Top 10% (2 melhores)
3. Identifique Bottom 10% (2 piores)
4. Proximo lote:
   - 70% baseado nos Top 2
   - 20% variacoes criativas
   - 10% experimental
5. Repita infinitamente
```
