# PARTE I: FUNDAMENTOS (Metodo Hormozi)

## A UNIDADE DE CONTEUDO

Todo conteudo eficaz segue esta formula:

```
HOOK (Fisgar) → RETER (Manter) → RECOMPENSAR (Entregar)
```

### 1. HOOK (Primeiros 3 segundos)
O hook e responsavel por 80% do sucesso do seu conteudo.

**Equacao do Hook Perfeito:**
```
HOOK = CALL OUT + PROMESSA DE VALOR
     = [Fazer a pessoa dizer "e pra mim"] + [Promessa se consumir]
```

### 2. RETER (Manter atenção)
Use uma destas 3 estruturas:
- **Listas:** "5 tecnicas que...", "3 erros que..."
- **Passos:** "Passo 1... Passo 2... Passo 3..."
- **Historias:** "Um dia eu estava... e entao..."

### 3. RECOMPENSAR (Entregar valor)
- Cumpra a promessa do hook
- Entregue mais do que prometeu
- Responda as perguntas que voce criou

**Regra de Ouro:** "Nao existe conteudo longo demais, apenas chato demais."

---

## OS 8 TIPOS DE HOOKS VERBAIS

| Tipo | Estrutura | Exemplo Breathwork |
|------|-----------|-------------------|
| **Rotulo** | "[Avatar], isso e pra voce" | "Ansiosos, tenho um presente" |
| **Pergunta Sim** | Pergunta com resposta obvia | "Voce quer dormir melhor?" |
| **Pergunta Aberta** | Gera curiosidade | "Qual voce preferiria ser?" |
| **Condicional** | "Se voce X, entao Y" | "Se esta estressado, esta respirando errado" |
| **Comando** | Instrucao direta | "Leia isso se quer parar de ter ansiedade" |
| **Declaracao** | Afirmacao forte | "A coisa mais poderosa que voce pode fazer hoje..." |
| **Lista/Passos** | Numero + promessa | "5 respiracoes que mudam sua vida" |
| **Narrativa** | Historia envolvente | "Eu estava no auge da ansiedade quando..." |

---

## 7 COMPONENTES DE MANCHETES QUE FUNCIONAM

Inclua pelo menos 2 em cada hook:

| Componente | Definicao | Exemplo |
|------------|-----------|---------|
| **Recencia** | O mais atual possivel | "Acabei de descobrir isso..." |
| **Relevancia** | Pessoalmente significativo | "Para quem sofre de ansiedade" |
| **Celebridade** | Pessoas proeminentes | "A tecnica que Navy SEALs usam" |
| **Proximidade** | Perto de casa | "Brasileiros estao respirando errado" |
| **Conflito** | Ideias opostas | "Respirar mais oxigenio NAO e melhor" |
| **Incomum** | Estranho, raro | "O que ninguem te conta sobre CO2" |
| **Em andamento** | Historia em progresso | "Estou testando isso ha 30 dias..." |

| Fonte | Descricao | Exemplo |
|-------|-----------|---------|
| **Passado Distante** | Licoes de vida | "Como superei minha ansiedade cronica" |
| **Passado Recente** | Ultimas semanas | "O que aprendi com meu ultimo aluno" |

---

## FRAMEWORK 70-20-10 PARA HOOKS

```
70% CORE (Replicar Vencedores)
├─ Use hooks que JA funcionaram (seus ou de outros)
├─ Adapte para seu contexto
└─ "Mais do que funciona"

20% EMERGENTE (Adjacente ao Vencedor)
├─ Variacoes dos vencedores
├─ Adapte conceitos de outros nichos
└─ Promissor mas menos provado

10% EXPERIMENTAL (Ideias Novas)
├─ Teste conceitos radicais
├─ Alto risco, alto retorno
└─ Fonte de breakthrough
```

**Regra:** Teste 10 hooks diferentes para cada conteudo importante.

---

## PROPORCAO DAR : PEDIR

```
MINIMO: 3:1 (3 posts de valor para 1 CTA de venda)
IDEAL: 10:1 ou mais (dar ate que pecam)
```

**Principio:** "De, de, de... ate que eles pecam."

## ESTRUTURA: CARROSSEL (7-10 slides)

### Modelo Padrao

```
SLIDE 1 - HOOK (80% do sucesso esta aqui!)
- Use um dos 8 tipos de hooks
- Pergunta provocativa OU
- Afirmacao surpreendente OU
- Problema que a audiencia tem

SLIDE 2 - CONTEXTO (Reter)
- Por que isso importa
- Estatistica ou dado [CIENTIFICO]
- Amplificar a dor/desejo

SLIDES 3-7 - CONTEUDO (Reter + Recompensar)
- Use LISTA, PASSOS ou HISTORIA
- 1 ideia por slide
- Maximo 30 palavras por slide
- Indicar fonte [CIENTIFICO] ou [HOLISTICO]

SLIDE 8 - RESUMO (Recompensar)
- Takeaway principal
- Frase memoravel
- Cumprir promessa do hook

SLIDE 9 - CTA
- Chamada para acao clara
- Proporcao 3:1 ou mais de dar
```

### Modelos Especificos

#### Modelo Lista (Ex: "5 tecnicas para...")
```
1. Hook: "5 respiracoes que mudam sua vida"
2. Intro: Por que respiracao importa [CIENTIFICO]
3-7. Uma tecnica por slide (LISTA)
8. Resumo visual
9. CTA: "Salve para praticar"
```

#### Modelo Problema-Solucao (HISTORIA)
```
1. Hook: O problema (ansiedade)
2. Por que acontece [CIENTIFICO]
3-4. O que piora o problema
5-6. A solucao (tecnica)
7. Como funciona [CIENTIFICO]
8. Resultado esperado
9. CTA
```

#### Modelo Mito vs Realidade (CONFLITO)
```
1. Hook: "Voce acredita nisso?"
2. Mito comum
3. Por que parece verdade
4. A realidade [CIENTIFICO]
5-6. Explicacao
7. O que fazer diferente
8. Resumo
9. CTA
```

## ESTRUTURA: REELS (15-60s)

### Modelo Padrao

```
[0-3s] HOOK (Pattern Interrupt!)
- Frase de impacto (use os 8 tipos)
- Parar o scroll IMEDIATAMENTE
- Visual ou sonoro inesperado
- Texto grande na tela

[3-15s] PROBLEMA/CONTEXTO (Reter)
- Identificar a dor
- Criar conexao
- "Voce tambem sente isso?"
- Use pergunta aberta

[15-45s] SOLUCAO (Reter + Recompensar)
- Tecnica ou insight
- Use PASSOS claros
- Demonstracao visual
- "Passo 1... Passo 2..."

[45-60s] CTA (Recompensar)
- Chamada para acao
- "Salva", "Manda", "Comenta"
```

### Modelos Especificos

#### Modelo Tutorial Rapido (30s)
```
[0-3s] COMANDO: "Faca isso quando estiver ansioso"
[3-25s] PASSOS: Demonstracao da tecnica 4-8
[25-30s] CTA: "Salva e testa agora"
```

#### Modelo Curiosidade (45s)
```
[0-3s] DECLARACAO CHOCANTE: "Voce respira ERRADO a vida toda"
[3-20s] PROBLEMA: Explicacao do problema
[20-40s] SOLUCAO: A solucao simples
[40-45s] CTA: "Segue pra mais"
```

#### Modelo Historia/Antes-Depois (60s)
```
[0-3s] NARRATIVA: "Antes eu vivia assim..."
[3-20s] Problema/situacao anterior
[20-40s] "Ai descobri breathwork..."
[40-55s] Transformacao
[55-60s] CTA emocional
```

## ESTRUTURA: YOUTUBE (5-15min)

### Modelo Padrao

```

- Use LISTA ou PASSOS
- Explicacao detalhada

- Passo a passo pratico

[12:00-13:00] RESUMO (Recompensar)
- Pontos-chave
- Cumprir promessa do hook
- Proximos passos

[13:00-15:00] CTA
- Inscreva-se
- Proximo video sugerido
- Link na descricao
```
